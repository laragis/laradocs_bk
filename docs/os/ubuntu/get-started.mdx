# Get Started
Check version
```shell
# Method 1
cat /etc/os-release

# Method 2
lsb_release -a

# Method 3
hostnamectl
```



## Package manager
### Cargo

```shell
# Install
apt update
apt-get -y install cargo

# Uninstall cargo, dependencies, configuration, data
apt-get -y autoremove --purge cargo
```

## Tips
### Uninstall package
```shell
apt-get -y autoremove --purge <PACKAGE>
```

### Switch user
```shell
# Switch to root user
sudo su

## Switch to user
su -l ubuntu

```
